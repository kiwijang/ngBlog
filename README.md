# NgBlog

用 angular 為前端搭配 Dendron 套件的部落格。
markdown 轉成 html 並可 deploy 到 Github Pages 上使用。

## 使用工具

- Nx.js
- Angular
- C#
- Dendron (VSCode Extension)
